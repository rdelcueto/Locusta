Detailed Description

template<typename TFloat>
struct locusta::de_solver_cpu< TFloat >

Forward declaration of the de_solver_cpu type.

CPU implementation of the differential evolution solver.

The de_solver_cpu class is a specific implementation of the evolutionary_solver for the CPU architecture, using the differential evolution algorithm.

This class implements the differential evolution solver for the CPU architecture.
