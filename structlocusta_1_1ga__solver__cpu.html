Detailed Description

template<typename TFloat>
struct locusta::ga_solver_cpu< TFloat >

Forward declaration of the ga_solver_cpu type.

CPU implementation of the genetic algorithm solver.

The ga_solver_cpu class is a specific implementation of the evolutionary_solver for the CPU architecture, using the genetic algorithm.

This class implements the genetic algorithm solver for the CPU architecture.
