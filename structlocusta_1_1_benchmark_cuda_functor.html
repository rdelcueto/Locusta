Detailed Description

template<typename TFloat>
struct locusta::BenchmarkCudaFunctor< TFloat >

CUDA evaluation functor for benchmark functions.

This class implements the EvaluationCudaFunctor interface for benchmark functions, enabling their evaluation on the GPU.
