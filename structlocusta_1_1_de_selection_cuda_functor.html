Detailed Description

template<typename TFloat>
struct locusta::DeSelectionCudaFunctor< TFloat >

Abstract base class for differential evolution selection operators on CUDA.

This class defines the interface for differential evolution selection operators on CUDA, which are used to select parents from a population.
