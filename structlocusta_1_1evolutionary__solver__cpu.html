Detailed Description

template<typename TFloat>
struct locusta::evolutionary_solver_cpu< TFloat >

Forward declaration of the evolutionary_solver_cpu type.

CPU implementation of the evolutionary_solver class.

The evolutionary_solver_cpu class is a specific implementation of the evolutionary_solver for the CPU architecture.

This class extends the evolutionary_solver class with CPU-specific functionality.
