Detailed Description

template<typename TFloat>
struct locusta::evolutionary_solver_cuda< TFloat >

Forward declaration of the evolutionary_solver_cuda type.

CUDA implementation of the evolutionary_solver class.

The evolutionary_solver_cuda class is a specific implementation of the evolutionary_solver for the CUDA architecture.

This class extends the evolutionary_solver class with CUDA-specific functionality.

Template Parameters
TFloatFloating-point type.
