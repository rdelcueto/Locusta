template<typename TFloat>
struct locusta::ga_solver_cuda< TFloat >

Forward declaration of the ga_solver_cuda type.

CUDA implementation of the genetic algorithm solver.

The ga_solver_cuda class is a specific implementation of the evolutionary_solver for the CUDA architecture, using the genetic algorithm.

This class implements the genetic algorithm solver for the CUDA architecture.
