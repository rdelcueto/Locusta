Detailed Description

template<typename TFloat>
struct locusta::pso_solver_cuda< TFloat >

Forward declaration of the pso_solver_cuda type.

CUDA implementation of the particle swarm optimization solver.

The pso_solver_cuda class is a specific implementation of the evolutionary_solver for the CUDA architecture, using the particle swarm optimization algorithm.

This class implements the particle swarm optimization solver for the CUDA architecture.
